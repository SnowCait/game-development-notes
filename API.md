## Request /Response
- ゲーム独自のステータスコードを用意する
  - エラー原因を特定できるようにするため
    - 数が増えるが各エラー原因で一意になるようにするのが望ましい
  - HTTP ステータスは全て 200 で返す
    - 正常系を 200 異常系を 500 で返してもいいかもしれない？
  - HTTP ステータス（３桁）や MySQL のエラーコード（４桁）と一目で区別できるように５桁にする
  - プレイヤーに見せるエラーコードも用意する
    - クライアント、サーバー、インフラ、プラットフォームなどが区別できるようにする
      - 英字＋数字にする
      - 数字の桁で区切る
